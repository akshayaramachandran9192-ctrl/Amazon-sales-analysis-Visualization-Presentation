# 📊 Data Analytics Project: End-to-End Power BI Dashboard

## 📌 Project Overview
This project delivers a comprehensive data analysis solution focused on transforming raw , unistructured E- commerce dataset into actionable business intelligence. Utilizing data from Kaggle, the project implements robust ETL processes, advanced data modeling, and interactive visualization techniques in Microsoft Power BI to uncover key performance indicators (KPIs) and critical business trends, empowering stakeholders with data-driven decision-making capabilities.

---

## 🛠️ Tech Stack & Tools
* **Data Source:** Kaggle Dataset
* **BI Tool:** Microsoft Power BI (Desktop & Service)
* **ETL & Transformation:** Power Query (M Language)
* **Data Modeling:** Star Schema Architecture (Fact and Dimension Tables)
* **Design & UI:** Customized Professional Theme Layout (`screenshot%2026-06-21%202548`)

---

## 🚀 Key Features & Development Phases

### 1. Data Cleaning & ETL (Power Query)
* Conducted rigorous data profiling to identify and handle missing values, anomalies, and duplicate records.
* Optimized data types and applied conditional columns to enrich the dataset for deeper analysis.

### 2. Data Modeling & DAX Implementation
* Formed a robust **Star Schema** relationship network between Fact and Dimension tables to ensure optimal report performance and accurate cross-filtering.
* Developed custom **DAX (Data Analysis Expressions)** measures for dynamic time-intelligence calculations and KPI tracking.

### 3. Interactive Visualizations & UI Design
* Built an intuitive, user-centric dashboard featuring high-level executive summaries and granular drill-down capabilities.
* Utilized structural design elements, consistent color palettes via custom JSON themes, and bookmarks for seamless navigation.
